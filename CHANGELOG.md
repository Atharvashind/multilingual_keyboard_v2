# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- Support for 6 languages: English, Hindi, Marathi, Telugu, Tamil, Bengali
- Vanilla JavaScript implementation with no dependencies
- Framework examples for React, Vue, and vanilla HTML
- Iframe embed option for CMS integration
- Text-to-speech functionality
- Responsive design for mobile and desktop
- Accessibility features (ARIA labels, keyboard navigation)
- Custom layout registration API
- Comprehensive documentation

### Features
- Language switching with one click
- Shift and Caps Lock support
- Physical keyboard integration
- Event system for integration
- Auto-injection of styles
- Mobile-optimized touch targets
